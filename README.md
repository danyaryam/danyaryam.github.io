# Portfolio Dany Arya Maulana

Website portofolio pribadi untuk menampilkan profil, pengalaman, keahlian, proyek, dan sertifikasi sebagai Frontend Engineer, Full-Stack Developer, dan AI Engineer.

## Live Website

[danyaryam.github.io](https://danyaryam.github.io/)

## Fitur

- Tampilan minimalis, responsif, dan modern
- Informasi pengalaman dan keahlian teknis
- Daftar proyek beserta peran dan teknologi
- Preview gambar proyek yang interaktif
- Daftar sertifikasi dan pembelajaran
- Navigasi responsif untuk desktop dan mobile

## Teknologi

- HTML5
- CSS3
- JavaScript
- GitHub Pages

## Menjalankan Secara Lokal

Clone repository ini, lalu buka `index.html` melalui browser atau gunakan ekstensi Live Server.

```bash
git clone https://github.com/danyaryam/danyaryam.github.io.git
cd danyaryam.github.io
```
